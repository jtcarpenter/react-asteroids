# React Asteroids

Simple game using React/Redux and react-konva for defining canvas objecs in JSX

## install

```
$ npm install
```

## Run (webpack)

```
$ npm start
```

## build prod (webpack)

```
$ npm run deploy
```

## tests

```
# Run once
$ npm test

# watch
$ npm run tdd
```
